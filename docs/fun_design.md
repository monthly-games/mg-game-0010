## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 아이템 제작/매입                                     │
│         ↓                                               │
│  2. 가격/배치 설정                                       │
│         ↓                                               │
│  3. 손님 방문/구매/반응 확인                             │
│         ↓                                               │
│  4. 수익으로 상점/던전 지원 확대 ─────────→ (1로 반복)    │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **아이템 제작/매입**: 판매할 아이템 제작 또는 도매 매입
2. **가격/배치 설정**: 전략적 가격 책정, 진열 배치
3. **손님 방문**: 손님 구매 행동 관찰, 피드백 수집
4. **상점/던전 확장**: 수익으로 상점 확장 및 던전 원정대 지원

## 3. 재미 축 (Fun Pillars)

### 3.1 경제 퍼즐 (Economy)
- **목표**: 수요/공급을 예측하고 최적 가격을 찾는 재미
- **핵심 경험**: 시장을 읽고 대박을 터뜨렸을 때의 쾌감
- **주요 요소**:
  - 수요/공급 시스템
  - 가격 탄력성
  - 경쟁 상점
  - 시장 이벤트

### 3.2 성장/확장 (Growth)
- **목표**: 작은 구멍가게가 대형 상점이 되는 성취감
- **핵심 경험**: 상점이 커지고 매출이 늘어나는 뿌듯함
- **주요 요소**:
  - 상점 면적 확장
  - 직원 고용
  - 품목 다양화
  - 시설 업그레이드

### 3.3 시장 읽기 (Prediction)
- **목표**: 시장 트렌드를 예측하는 전략적 재미
- **핵심 경험**: 내 예측이 맞았어! 하는 성취감
- **주요 요소**:
  - 트렌드 분석 도구
  - 시즌 이벤트
  - 뉴스/이벤트 힌트
  - 투자 시스템

## 4. 디자인 레버 (Design Levers)

### 4.1 경제 시각화
| 레버 | 설명 |
|------|------|
| 수요/공급 곡선 시각화 | 아이템별 수요/공급 곡선을 시각화하여 경제 퍼즐화 |
| 구현 방향 | 아이템 상세 페이지에 수요 그래프, 가격 추천 시스템 |

### 4.2 손님 세그먼트
| 레버 | 설명 |
|------|------|
| 세그먼트별 차별화 | 손님 세그먼트(초보/고급/모험가 등)마다 선호/민감도 차별 |
| 구현 방향 | 초보 모험가 → 저가 포션, 베테랑 → 고급 장비, VIP → 한정판 |

### 4.3 던전 연계
| 레버 | 설명 |
|------|------|
| 상점→던전 영향 | 상점 판매 품목이 던전 원정대 성과에 영향을 미치게 설계 |
| 구현 방향 | 회복약 재고 풍부 → 원정대 생존율 UP, 무기 판매 → 원정대 공격력 UP |

### 4.4 이벤트 타이밍 게임
| 레버 | 설명 |
|------|------|
| 수요 폭등 구조 | 이벤트/시즌별로 특정 아이템 수요 폭등 구조로 타이밍 게임 제공 |
| 구현 방향 | 드래곤 출현 이벤트 → 화염 방어구 수요 300%, 축제 → 선물 아이템 폭등 |
