## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 경제 시스템 안정화 |
| 7~10주 | 튜토에서 "가격 조정 성공 경험" 제공 |
| 11~12주 | 첫 시즌 이벤트 준비 |

**핵심 과제:**
- 경제 시스템 밸런스
- 가격 조정의 재미 체감
- 튜토리얼 성공 경험

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 축제 이벤트 (수요 급증) |
| 5개월 | 세일 기간 이벤트 |
| 6개월 | 던전 원정 연계 이벤트 |
| 7개월 | 특수 고객층 등장 |
| 8개월 | 여름 대축제 + 원정 보상 변화 |

**핵심 과제:**
- 시즌 이벤트 경제 영향 검증
- 던전 연계 시스템
- 수요/공급 이벤트 사이클

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 콜라보 상품/브랜드 아이템 실험 |
| 10개월 | 다른 게임과 아이템 교차 이벤트 |
| 11개월 | 대규모 시장 충격 이벤트 (전쟁) |
| 12개월 | 1주년 대축제 이벤트 |

**핵심 과제:**
- 콜라보 상품 검증
- 크로스 게임 아이템 연동
- 대규모 경제 이벤트

## 4. 이벤트/운영 패턴
