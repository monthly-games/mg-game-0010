> **문서 버전**: 1.0.0

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 제작 시작 | 슬롯 영역 | 탭 |
| 제작 완료 | 완료 버튼 | 탭 |
| 판매 | 아이템 → 손님 | 드래그 |
| 레시피 보기 | 제작 버튼 | 롱프레스 |

### 터치 피드백
- 제작 시작: 슬롯 활성화 애니메이션
- 제작 완료: 아이템 팝 + 파티클
- 판매 성공: 골드 플라이 이펙트
