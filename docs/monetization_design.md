## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `sale_boost` | 판매 시 | 판매액 50% 추가 | 무제한 |
| `craft_speed` | 제작 대기 | 즉시 완료 | 10회 |
| `customer_vip` | VIP 손님 | VIP 손님 즉시 방문 | 3회 |
| `supply_bonus` | 공급 구매 | 20% 할인 | 5회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |
| `dungeon_loot` | 던전 보상 | 드롭 2배 | 3회 |

### 1.2 전면 광고

- **트리거**: 영업 마감 시
- **쿨다운**: 5분
- **구독자 제외**
