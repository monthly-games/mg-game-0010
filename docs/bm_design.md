## 3. 상세 BM 설계

### 3.1 꾸미기/상점 확장 BM

#### 무료 구조
- 스토리 진행으로 기본 인테리어 획득
- 레벨업으로 상점 면적 자동 확장
- 업적 달성으로 특수 장식 해금

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 상점 테마 세트 | $4.99~$9.99 |
| 단품 장식 | 개별 프리미엄 장식 | $0.99~$2.99 |
| 상점 확장권 | 즉시 면적 확장 | $2.99~$4.99 |
| 프리미엄 진열대 | 특수 기능 진열대 | $1.99~$3.99 |
| 간판/외관 | 상점 외부 장식 | $2.99 |

### 3.2 방치 수익 BM

#### 무료 구조
- 오프라인 판매 상한: 10시간
- 자동 가격 조정 시스템
- 일일 접속 보너스

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 판매 수익 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 판매 수익 3배 패스 | 상시 적용 (30일) | $3.99/월 |
| 오프라인 확장 | 20시간까지 누적 | 패스 포함 |
| 자동 재고 보충 | 품절 자동 보충 | $1.99 (영구) |

### 3.3 경영 편의 BM

#### 무료 구조
- 기본 레시피/제작 무료
- 던전 원정대 기본 파견 가능
- 시장 정보 기본 제공

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 시장 분석 도구 | 수요/공급 예측 | $2.99 |
| 직원 슬롯 | 추가 직원 고용 | $1.99 |
| 창고 확장 | 재고 보관량 증가 | $1.99~$3.99 |
| 원정대 슬롯 | 추가 원정대 파견 | $2.99 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 판매 2배, 희귀 재료, 시장 힌트 |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금으로 접근 가능
- 경영 성과는 전략이 과금보다 중요
- 꾸미기 아이템 기능성 버프 최소화

### 과금 압박 최소화
- 경쟁 요소 없는 개인 시뮬레이션
- 시간이 지나면 자연스럽게 성장
- 광고로 핵심 자원 충분히 획득 가능
